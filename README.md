# DecodeLabs AI Internship — Project Portfolio (Batch 2026)

Solutions for the four DecodeLabs Artificial Intelligence Industrial Training Kit projects.

## Projects

| # | Folder | Project | Status |
|---|--------|---------|--------|
| 1 | [project-1-rule-based-chatbot](project-1-rule-based-chatbot/) | Rule-Based AI Chatbot | Mandatory |
| 2 | [project-2-data-classification](project-2-data-classification/) | Data Classification Using AI (Iris + KNN) | Mandatory |
| 3 | [project-3-tech-stack-recommender](project-3-tech-stack-recommender/) | AI Recommendation Logic (Tech Stack Recommender) | Mandatory |
| 4 | [project-4-image-text-recognition](project-4-image-text-recognition/) | Image or Text Recognition (OCR) | Optional |

Each folder contains its own README with the project brief, how to run it, and how it meets the qualification criteria.

## Requirements

- Python 3.10+
- `pandas`, `numpy`, `scikit-learn`, `matplotlib` (Projects 2 & 3)
- `opencv-python`, `pytesseract` + the Tesseract engine (Project 4 only)

```bash
pip install pandas numpy scikit-learn matplotlib opencv-python pytesseract
sudo apt install tesseract-ocr        # Project 4 only
```

## Quick start

```bash
python3 project-1-rule-based-chatbot/chatbot.py
python3 project-2-data-classification/classification.py
python3 project-3-tech-stack-recommender/recommender.py
python3 project-4-image-text-recognition/ocr.py
```
